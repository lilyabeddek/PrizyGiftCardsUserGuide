Cartes cadeaux Prizy : Mode d’emploi

À propos de nos cartes cadeaux Prizy

Les cartes cadeaux Prizy offrent à leurs bénéficiaires la possibilité de les utiliser dans un vaste réseau de partenaires, couvrant diverses catégories telles que le prêt-à-porter, les accessoires, l'électroménager, l'électronique, le sport, le bien-être, et l'agroalimentaire. De plus, elles peuvent être utilisées sur les sites web partenaires répertoriés dans notre catalogue.

Vous avez la flexibilité d'utiliser votre carte cadeau en plusieurs fois, dans différentes enseignes, tant que son solde n'est pas épuisé et sa date de validité n'est pas dépassée.

Informations générales sur les cartes cadeaux Prizy

Canaux de réception d’une carte cadeau Prizy

Les cartes cadeaux digitales de Prizy ont été spécialement développées pour être associées à l'adresse électronique du bénéficiaire. Vous recevrez alors votre carte cadeau par e-mail, envoyée par l'entreprise ou la personne qui souhaite vous l'offrir. Cependant, une fois que vous l'avez reçue par courrier électronique, vous avez également la possibilité de l'imprimer si vous le désirez.

Connaître le montant total de ma carte cadeau Prizy

Lorsque vous recevez l'e-mail contenant votre carte cadeau Prizy, l'un des éléments fournis est le montant de celle-ci. Vous pourrez facilement repérer cette information dans le coin supérieur gauche de l'e-mail, juste sous votre nom, comme illustré dans l'exemple ci-dessous.

Connaître le type de ma carte cadeau Prizy

Prizy offre une variété étendue de cartes cadeaux (voir nos offres) qui se distinguent les unes des autres en fonction du groupe d'enseignes où elles peuvent être utilisées.

Pour identifier le type de carte cadeau que vous avez reçu, il vous suffit de consulter l'image présente dans l'e-mail, juste après le montant total de la carte, comme le montre l'exemple ci-dessous :

Si votre logiciel de messagerie n'affiche pas les images par défaut (par exemple, Outlook), veuillez le configurer manuellement pour visualiser tous les détails de la carte cadeau correctement.

Connaître la date de validité de ma carte cadeau Prizy

Nos cartes cadeaux possèdent une caractéristique importante : leur validité est définie sur une période donnée, entre une date de début de validité (avant laquelle la carte ne peut être utilisée) et une date d'expiration (après laquelle la carte n'est plus utilisable).

Ces dates spécifiques sont clairement indiquées dans l'e-mail contenant la carte cadeau, juste après l'image qui identifie le type de carte, comme illustré dans la figure précédente.

Informations de paiement de ma carte cadeau Prizy

Pour effectuer un paiement avec une carte cadeau Prizy, vous aurez uniquement besoin de deux informations :

  1. La clé de la carte cadeau : Celle-ci est disponible sous deux formats différents, soit sous forme de QR code, soit sous format texte. Vous devrez présenter cette clé au caissier du magasin, qui la scannera ou la saisira en fonction du format fourni.
  2. Le code secret : Ce code à 4 chiffres vous sera demandé par le vendeur pour finaliser votre transaction.

Comment utiliser une carte cadeau Prizy ?

Où consommer ma carte cadeau Prizy ?

En fonction du type de carte cadeau que vous avez, vous pourrez l'utiliser dans un certain nombre de magasins ou de sites web partenaires affiliés à Prizy. Pour consulter la liste complète de ces partenaires, il vous suffit de cliquer sur le bouton "Enseignes partenaires" situé en bas de votre e-mail, juste en dessous de la clé de la carte cadeau, comme le montre l'illustration ci-dessous :

En cliquant sur ce bouton, vous serez redirigé vers une nouvelle page qui présente les partenaires affiliés. En choisissant l'une des enseignes de la liste, vous accéderez à une autre page qui répertorie les magasins de cette enseigne où vous pourrez utiliser votre carte, ainsi que leurs horaires d'ouverture et leur emplacement. Si le magasin propose également des ventes en ligne, un lien vers leur site web vous permettra d'utiliser votre carte sur leur plateforme.

Remarque

Veuillez noter que certaines enseignes peuvent avoir plus de magasins que ceux indiqués dans la liste. Dans ce cas, votre carte ne pourra être utilisée que dans les magasins spécifiés.

Comment effectuer un paiement avec ma carte cadeau Prizy ?

Pour effectuer un paiement avec votre carte cadeau Prizy, veuillez suivre les instructions décrites dans la section précédente.

Est-il possible d'utiliser mes cartes cadeaux Prizy en ligne ?

Pour utiliser votre carte cadeau dans l'une de nos enseignes partenaires, sélectionnez simplement l'enseigne de votre choix dans la liste. Si vous repérez une rubrique "site web" avec un lien vers le site correspondant, alors vous pourrez effectivement utiliser votre carte cadeau en ligne. En revanche, si cette rubrique n'est pas présente, malheureusement, l'utilisation en ligne ne sera pas possible.

Remarque

Veuillez noter qu'il peut arriver qu'une de nos enseignes partenaires propose des ventes en ligne via son site web, mais que vous ne le trouviez pas dans la liste des magasins (comme c'est le cas par exemple avec Zara). Dans cette situation, regrettablement, votre carte cadeau ne pourra pas être utilisée en ligne. C'est pourquoi il est toujours essentiel de vérifier la liste des magasins pour obtenir des informations à jour.

Comment utiliser ma carte cadeau Prizy en ligne ?

Pour utiliser votre carte cadeau pour un paiement en ligne, suivez ces étapes simples :

  1. Assurez-vous tout d’abord que l’utilisation de la carte cadeau est possible en ligne (voir question précédente)
  2. Procéder à votre achat en ligne comme d’habitude
  3. Lorsque l’enseigne vous contacte pour confirmer votre commande, indiquez clairement que vous souhaitez utiliser Prizy comme moyen de paiement
  4. Suite à cela vous recevrez une demande de paiement par e-mail, où vous devrez saisir les informations de paiement de votre carte cadeau, notamment la clé et le code secret
  5. Une fois la demande de paiement validée, vous obtiendrez la confirmation de votre commande

Comment faire le suivi de mes achats faits avec ma carte cadeau Prizy ?

À chaque fois que vous effectuez un achat en utilisant votre carte cadeau, vous recevez un e-mail récapitulatif contenant les détails de votre transaction, tels que la date, le lieu de l'achat, le montant dépensé et le solde restant sur votre carte cadeau.

Comment consulter le solde de ma carte cadeau Prizy ?

Si vous n'avez pas encore utilisé votre carte cadeau, vous pouvez vérifier le solde initial de celle-ci en consultant l'e-mail contenant la carte cadeau que vous avez reçu.

En revanche, si vous avez déjà effectué des achats avec la carte, pour connaître le solde restant sur celle-ci, il vous suffit de consulter le dernier e-mail de confirmation d'achat que vous avez reçu lors de votre dernière transaction.

Est-il possible d’utiliser la carte cadeau Prizy pendant les offres/promotions ?

Pendant les soldes, l'utilisation d'une carte cadeau Prizy suit les règles de vente de chaque magasin. Il est recommandé de contacter directement le magasin qui vous intéresse avant de vous déplacer, pour vous assurer des détails. Leurs coordonnées se trouvent sur leurs pages spéciales, dans notre espace enseignes (voir figure).

En cas de confusion ou de problème, n'hésitez pas à contacter notre service clientèle. Vous trouverez nos coordonnées en bas de l'e-mail contenant votre carte cadeau. Notre équipe se fera un plaisir de vous aider.

Que dois-je faire en cas de retour ou d'échange produit dans une des enseignes partenaires ?
